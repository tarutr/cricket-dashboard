  Columns picker, player search / pins, column-header SORT.

  Search button: enabled+highlighted(is-dirty) iff (dirty && searchable); else
  disabled — mirrors graph Update-chart btn + the searchable gate.

  LOAD now, not first Search); syncToolbar() re-syncs values/enable/dirty/count

  RIGHT [count · SEARCH · Columns · Clear]. Graph btn REMOVED.

  enterFromBridge no longer lands on Dumbbell/consumes the key since R2 — flag.)
